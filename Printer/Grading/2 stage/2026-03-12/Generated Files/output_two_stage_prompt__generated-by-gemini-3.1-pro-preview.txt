            state Idle
            state PrintReady
            state Printing
            state PrintSuspendedJam
            state PrintSuspendedPaper
            state ScanReady

            state ScanSuspendedJam

            Idle --> ScanReady : chooseScan

            Printing --> PrintSuspendedJam : paperJam

            PrintSuspendedPaper --> Printing : resume
